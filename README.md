## About The Project

### Goals
I wanted to play around with the new version of Blazor and .NET 8. 

### Built With
* [Blazor][Blazor-url]
* .NET 8
## Getting Started

### Prerequisites

* .NET 8
* Visual Studio 2022 (I am using the preview version)

### Installation

1. Download Visual Studio 2022
2. Clone the repo
   ```sh
   git clone https://github.com/mlottes/BlazorPinballLocations
   ```
3. Open the solution
4. Press play
